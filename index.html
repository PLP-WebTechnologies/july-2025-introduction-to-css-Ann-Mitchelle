About This Page

This page demonstrates basic CSS styling and how the Box Model works. We will use margin, padding, and borders to adjust spacing and layout.

The CSS Box Model

The Box Model consists of:

  • Content: The text or image inside the element.
  • Padding: Space between content and border.
  • Border: The line surrounding the padding.
  • Margin: Space outside the border.
